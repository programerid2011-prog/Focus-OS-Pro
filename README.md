# Focus OS Pro: Ultimate CBSE Command Center
A high-performance, unified study dashboard engineered for Class 10 Board Exam preparation (2026-27).
Read the full engineering documentation and feature breakdown on my [Technical Blog here](https://harsh-dev.blogspot.com/2026/04/focus-os-pro.html).

## 🚀 Purpose
This project was designed to solve the problem of "tab-fatigue" and fragmented study tools. I built a single-interface "OS" that combines time management, accountability, and active recall.

## 🛠️ Key Features
- **Integrated Pomodoro Hub:** Custom timer with a built-in Focus Audio Engine (Brown Noise & Binaural Beats).
- **Target 95% Flashcard System:** A custom-built active recall module for all CBSE subjects (Science, Maths, SST, IT, etc.).
- **Dynamic Accountability Calendar:** Automatically tracks and visualizes study distribution and consistency.
- **Hardcore Mode:** A technical constraint that penalizes XP if the user leaves the study tab, enforcing discipline.
- **Data Persistence:** Full LocalStorage integration with a custom JSON Backup/Restore engine.

## 💻 Technical Stack
- **Frontend:** HTML5, CSS3 (Custom Variables/Grid/Flexbox)
- **Logic:** Vanilla JavaScript (ES6)
- **Visualization:** Chart.js
